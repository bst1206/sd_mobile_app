Detailed Description

Toggles LED0 when button pressed.

Configures pin controlling LED0 and LED1 to be an output. Configures pin attached to button as an input with interrupt. Once per second toggles LED0 as a simple indication that application is running. When interrupt generated, waits to allow button to debounce then if button pressed, toggles LED0.

This example is a good way to verify that the development environment is configured correctly.

http://processors.wiki.ti.com/index.php/Tutorial_on_the_Examples and http://e2e.ti.com/support/low_power_rf/default.aspx
