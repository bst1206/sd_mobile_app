Detailed Description

Resets Radio, configures this device to be a Zigbee Coordinator, and displays any messages that are received. Uses the AF/ZDO interface.

This matches example_basic_communications_ROUTER.c Get this running before the router, or else the router won't have anything to join to.

Basic Coordinator Startup:

  • Reset Radio
  • Set Startup Options = CLEAR_STATE and CLEAR_CONFIG - This will restore the ZNP to "factory" configuration
  • Reset Radio
  • Set Zigbee DeviceType to COORDINATOR
  • Note: If you want to set a custom PANID or channel list, do that here
  • Register Application (Configure the ZNP for our application)
  • Start Application
  • Wait for the Start Confirm
See also:
