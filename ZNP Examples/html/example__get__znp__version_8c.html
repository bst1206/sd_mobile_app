Detailed Description

Resets Radio, gets ZNP version information using SYS_VERSION command and displays it.

Configures the MSP430 to communicate with the radio, resets the radio, and sends the SYS_VERSION command and parses the received response and displays it to the console. Demonstrates basic request-response functionality of the ZNP.
