Detailed Description

Resets Radio, prompts the user for which NV item to write (1-6) and writes test data to that location.

The ZNP supports six NV storage buffers (named "NV Items"). 1-4 are 2 bytes each; 5 & 6 are 16 bytes each. These NV items are stored in the ZNP's flash memory and can be used for non-volatile storage.

See also:
http://processors.wiki.ti.com/index.php/Tutorial_on_the_Examples and http://e2e.ti.com/support/low_power_rf/default.aspx
